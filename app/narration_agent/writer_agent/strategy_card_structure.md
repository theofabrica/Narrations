# strategy_card_structure.md - Field clarification

The `strategy_card_structure.json` file describes the writing strategy selected
after searching the library.

## Fields
- `strategy_id`: short identifier of the chosen strategy.
- `target_path`: path of the target field (e.g. `core.summary`).
- `writing_typology`: writing task type used for strategy selection.
- `library_item_ids`: ids of library items used as sources.
- `style_guidelines`: style guidelines (short list).
- `structure_guidelines`: structure guidelines (order, rhythm, length).
- `source_refs`: references to source texts (title + short excerpt).
- `notes`: useful notes (optional).

## Rules
- The strategy must be justified by sources present in the library.
- Guidelines must remain actionable and concise.
